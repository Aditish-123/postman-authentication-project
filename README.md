# Token-Based Authentication API Testing

This project demonstrates a complete authentication flow using Postman and ReqRes API.

## Features
- Login API testing
- Token extraction
- Environment variables
- API chaining
- Bearer token authentication
- Collection Runner execution

## Tools Used
- Postman
- REST API
- JavaScript
- ReqRes API

## Project Flow
1. Login API request
2. Extract token dynamically
3. Store token in environment variable
4. Use token in GET request
5. Validate response
